Den 4: Další krok na cestě web developmentem: První životopis a spletitý svět CSS

Ach jo, další den na cestě za webem. Dnes jsem přišla o svoji "web development" nevinnost a vytvořila svůj první životopis online. Povedlo se! Musím říci, že už trochu začínám chápat toto kouzelné umění.

Konečně jsem zjistila, co to jsou ty CSS (ne, není to zkratka pro Cool Super Star). Ukázalo se, že existují tři různé typy - Inline, Internal a External. Je to jako s trikem - taky je máme v různých stylech a podle nálady a potřeby si vybíráme ten správný.

Zatím se mi ale ty styly stále pletou... Snad za pár dní budu ladit CSS jako virtuos na housle. Pravda, vývoj webových stránek může být svým způsobem i hudba!

Takže pokud mě hledáte, jsem nejspíš někde zahrabaná pod hromadou HTML a CSS kódů, hledám cestu ven pomocí stylů a divů. Ale jak říkám, svět web developmentu je krásný... Alespoň tak mi bylo řečeno. Uvidíme, jestli se mi podaří najít tu jeho krásu. Pokud má někdo mapu, tak ji prosím určitě pošlete!
