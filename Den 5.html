Den 5: CSS - Mé taneční kroky mezi tagy

Tečka, čárka, stvořena další stránka! S CSS jsem se pustila do duše své webové krásky a ponořila se do nástrah barevné palety a srovnávání kontrastů mezi html tagem a body tagem. A hle, je to jako porovnávat jablka a... ach, červená jablka.

Bravičky, fonty a spousta dalších puntíkatých koláčků samozřejmě doprovázejí tento barevný balet. Ale tvrdohlavá strana mě nabádá, abych se vrhla do bitevní vřavy pokusů. Identita webovek se nevytváří přes noc!
